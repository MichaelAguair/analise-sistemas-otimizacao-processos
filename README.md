# analise-sistemas-otimizaçao-processos-de-aquisição
Projeto com objetivo de Adquirir novos clientes onde um time de Atendimento possa estar participando processo na jornada de um novo cliente, a viabilidade de efetivar um acompanhamento da jornada do cliente / lead, onde é detectado dentro dos modelos de atendimento do  um encaminhamento para canais que executam a ação necessária. Após um acompanhamento, métricas e indicadores do resultado gerado  para ser visualizado. Dessa forma o projeto tem intuíto de  tangibilizar essas efetivações e acompanhamento dos resultados gerados.

Projeto de Análise de Sistemas — Otimização de Processos Operacionais IAA (Iniciativa Atendimento Aquisição)

-- Contexto
Este projeto é um estudo de caso baseado em um cenário real, com dados totalmente anonimizados e fictícios, com objetivo de demonstrar habilidades em análise de sistemas, levantamento de requisitos e melhoria de processos.

-- Problema de Negócio
O projeto nasceu da identificação de um processo onde o atendimento ao cliente influênciar em vendas junto ao comercial. O principal obstáculo detectado foi o conflito de horários do Atendimento 24/7 e Vendas apenas no horário Comercial

-- Análise Realizada
Esse problema detectado fazia com que o a jornada do cliente para aquisições e possíveis leads afetassem indicadores importantes, dessa forma um fluxo com atuações seria necessário para ajuste de aquisições e satificação de clientes.

-- Requisitos e Solução Proposta
Desenvolver uma estratégia para preparar o cliente antes mesmo do contato inicial. O objetivo é escalar os casos com eficiência através da coleta prévia de dados essenciais e da criação de uma conexão sólida e confiável com o cliente desde o primeiro momento.
